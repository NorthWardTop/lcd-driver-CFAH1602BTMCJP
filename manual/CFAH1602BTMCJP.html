10.Function Description 10.功能描述
The LCD display Module is built in a LSI controller, the controller has two 8-bit registers, an instruction LCD显示模块内置在LSI控制器中，控制器有两个8位寄存器，一条指令
register (IR) and a data register (DR). 寄存器（IR）和数据寄存器（DR）。
The IR stores instruction codes, such as display clear and cursor shift, and address information for display data IR存储指令代码，例如显示清除和光标移位，以及显示数据的地址信息
RAM (DDRAM) and character generator (CGRAM). RAM（DDRAM）和字符发生器（CGRAM）。 The IR can only be written from the MPU. IR只能从MPU写入。 The DR DR
temporarily stores data to be written or read from DDRAM or CGRAM. 临时存储要从DDRAM或CGRAM写入或读取的数据。 When address information is 当地址信息是
written into the IR, then data is stored into the DR from DDRAM or CGRAM. 写入IR，然后将数据从DDRAM或CGRAM存储到DR中。 By the register selector (RS) 通过寄存器选择器（RS）
signal, these two registers can be selected. 信号，这两个寄存器可以选择。
